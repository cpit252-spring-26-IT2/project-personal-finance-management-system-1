# Project Name
Personal Finance Management System
## Description
A Java-based application designed to help users manage their personal finances in one 
place. The system allows users to track expenses and income, monitor their savings, and 
receive basic investment advice based on their financial situation. It simplifies financial 
decision-making and promotes better money management habits.
